# Abacum Integration

Abacum is a financial planning and analysis (FP&A) platform that helps finance teams build budgets, create forecasts, and generate collaborative financial reports. Light connects with Abacum via S3 bucket. The data share setup is prepared by Light, and once ready, Abacum connects automatically.

[Open in Light →](https://app.light.inc/settings/integrations)

## Setting up your Light connection

1. Contact your Light representative to request data sharing with Abacum
2. The Light team will provide the necessary credentials (**Access key ID**, **Secret access key**, **S3 Bucket name**)
3. Navigate to the **Integrations** section in Abacum
4. Select Light via S3 and click **Connect**
5. Input the **Access key ID**, **Secret access key**, and **S3 Bucket name** that you received from Light
6. Click on **Connect to Amazon S3** to complete the setup

## Create a new dataset

Once the connection is established, you can import your Light ERP data into Abacum:

1. Select the S3 connection within the **Create Dataset** dialog
2. Specify your S3 view name and select the synchronization month
